# [11장] 컨테이너 자료구조 (TMap) 예상 문제

## 문제 1
다음은 `TMap`을 사용하여 키(Key) 값으로 아이템 이름을 검색하는 코드이다. 코드의 빈칸 (A)에 들어갈 올바른 기호를 고르시오.

```cpp
TMap<int32, FString> InventoryDict;
InventoryDict.Add(1, TEXT("Sword"));

// Find 함수는 값을 복사해서 주지 않고 메모리 주소(포인터)를 반환합니다.
if (FString[   (A)   ] FoundItem = InventoryDict.Find(1))
{
    UE_LOG(LogTemp, Warning, TEXT("아이템 발견!"));
}
```

1) `&` (참조자)
2) `*` (포인터)
3) `< >` (템플릿)
4) 아무 기호도 필요 없음 (값 복사)

---
### 정답 및 해설
**정답: 2번 (*)**
**해설:** 언리얼 `TMap`의 `Find()` 함수는 해당 키가 딕셔너리에 존재하지 않을 경우 에러(Crash)를 내지 않고 안전하게 `nullptr`을 반환할 수 있도록 설계되어 있습니다. 따라서 반환형은 반드시 포인터(`*`) 타입이어야 하며, `if`문 안에서 포인터 널 체크를 동시에 수행하는 것이 C++의 정석입니다.
