# [10장] 게임 수학과 벡터 (Vector) 예상 문제

## 문제 1
플레이어 폰(`PlayerPawn`)과 적 폰(`EnemyPawn`)이 있다. 플레이어가 쏜 총알이 적의 정면(앞통수)을 맞혔는지, 아니면 암살처럼 뒤에서(뒤통수) 맞혔는지 C++ 코드로 판별하려고 한다. 
플레이어가 쏜 방향 벡터(`HitDirection`)와 적의 정면 방향 벡터(`EnemyForward`)를 가져와 **내적(Dot Product)** 연산을 수행했다. 
만약 **뒤에서(뒤통수) 맞혔다(두 벡터의 방향이 거의 같다)**고 판별되려면 내적 결과값(`DotResult`)의 범위는 어떠해야 하는가?

1) `DotResult > 0`
2) `DotResult == 0`
3) `DotResult < 0`
4) `DotResult == 1`

---
### 정답 및 해설
**정답: 1번 (DotResult > 0)**
**해설:** 총알이 날아가는 방향(HitDirection)과 적이 바라보는 방향(EnemyForward)이 **같은 쪽**을 향하고 있다면 사잇각이 예각(90도 미만)이 되므로 내적(Cos) 값은 **양수(>0)**가 나옵니다. 반면 앞통수를 맞았다면 두 벡터가 마주보는(반대) 방향이므로 둔각이 되어 내적 값이 음수(<0)가 나옵니다.
