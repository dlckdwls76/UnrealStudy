# [6장] 충돌(Collision) 시스템 예상 문제

## 문제 1
다음 코드는 런타임 중에 플레이어 액터의 위치를 `FVector(1000, 0, 0)`으로 강제 이동(텔레포트)시키는 코드이다.

```cpp
SetActorLocation(FVector(1000, 0, 0), true);
```

이때 두 번째 매개변수(Parameter)로 전달된 **`true` (bSweep)** 옵션의 역할로 가장 올바른 것은?

1) 이동하는 즉시 해당 좌표의 이펙트 입자를 쓸어(Sweep)버린다.
2) 이동 과정에서 중간 경로에 다른 Block 판정의 액터(벽, 장애물)가 있다면, 통과하지 못하고 벽에 부딪혀 멈추게 만든다.
3) 벽을 무시하고(통과하고) 무조건 `(1000, 0, 0)` 위치로 순간이동시킨다.
4) 텔레포트 시 발생하는 물리적 관성(Velocity)을 초기화한다.

---
### 정답 및 해설
**정답: 2번**
**해설:** 언리얼의 `Sweep` 옵션을 `true`로 켜면, 시작 지점부터 목표 지점까지 액터의 콜리전을 빗자루 쓸듯 쭉 밀어보면서 충돌 검사를 수행합니다. 만약 중간에 막히는 물체가 있다면 벽 뚫기(통과)를 방지하고 그 자리에 멈춰 서게 됩니다.
